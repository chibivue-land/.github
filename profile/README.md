## 定義

chibivue land は日本の Vue.js コミュニティを起点とした，インターネット上に存在する国です．

(Vue.js および Vue.js JP が公式に運営しているものではありません．非公式コミュニティです)

住所: https://discord.gg/aVHvmbmSRy

Web サイト: https://chibivue.land

この国の国王は [@ubugeeei](https://github.com/ubugeeei) として広く知られていますが，もののけの王として祟り神になりました．\
※ 稀にインターネット上に人間の形としてその姿をあらわにすることがあります．

今は，[@karacoro](https://github.com/tsukuha) と [@kazupon](https://github.com/kazupon) を王の影武者とし，国の存続に尽くしています．\
その他に，国王が存命だった頃に側近としての役割を与えられた人類が数人存在します．\
彼らは国王自らが推薦した，信頼を置いているメンバーであり，王座があいた今でも歴史と文化を尊重し，国の発展に励んでいます．

この国では絶対王政を採用しており，明確な行動規範は存在しません．\
管理者 (影武者または側近) が不適切だと判断したコミュニケーションや投稿，人物などに対しては無告知で削除/強制退出を実施する可能性があります．(滅多にありません)

### 納税の義務

来訪者ではなく，国民として扱われるには納税の義務があります．\
税金を納める方法はいくつかあります．

納税の形:
- 国の発展に貢献する (国を盛り上げる)
- コミュニティや OSS に貢献する．

といっても，ハードルを感じる必要はありません．\
どんなに些細な活動であっても，どのような形でもこの国では納税義務を果たしたことになります．\
それぞれが考える，定義で問題ありません．(前述の通り，不適切な行動に対しては権限を持った者から追放される可能性はあります．)

## 性質

この国では，OSS に関する話や，仕事の話，他愛もない雑談が盛んであり，誰でも気軽に入国することが可能です．

また，この国には「広場」と「住宅街」と呼ばれる場所が存在しており，広場では多様なトピックグループがありつつも希望者は住宅街に自身の家屋を持つことができます．\
家屋といっても屋根がある補償はありません．また稀に放火魔が現れることもあります．\
自分の家が欲しい方はリクエスト用のチャンネルがあるので，管理者に申し出てください．\
建築費用はこの国の税金によって賄われるため，無料です．\
税金を納めている全ての国民に感謝しましょう．
